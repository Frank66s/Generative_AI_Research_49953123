## Folder Descriptions

### Literature_Review

This folder contains academic sources used to support the background research.  
These files include journal articles and other readings about generative AI, ChatGPT in education, and AI ethics.

### Quantitative_Analysis

This folder contains numerical and structured research data.  
It includes survey data, survey questions, CSV files, Excel files, Python analysis scripts, and quantitative data visualisations.

For example, the scatter plot shows the relationship between participants' score after the activity and their perceived learning outcome level.

### Qualitative_Analysis

This folder contains non-numerical research data.  
It mainly includes interview transcripts from different participants. These files can be used for coding, theme analysis, and identifying participants' opinions and experiences.

### Drafts_and_Reports

This folder contains draft versions, final reports, and other written research documents.  
It is used to keep track of the development of the research report.

### Additional_Materials

This folder contains supporting materials, images, screenshots, diagrams, and other media files.  
These files help explain the project structure, workflow, and research process.

Use `Literature_Review` to find academic sources and background readings.

## Collaboration Guidelines

Future collaborators should follow these steps when contributing to this repository:

1. Pull the latest version of the repository before making changes.
2. Create or edit files in the correct folder.
3. Use clear and meaningful file names.
4. Write a short but meaningful commit message.
5. Push the changes to GitHub after committing.
6. If working in a group, use branches and pull requests before merging changes into the main branch.
7. Check for conflicts before merging changes.

Example commit messages:

```text
Added cleaned survey data
Updated interview transcript files
Added quantitative analysis visualisation
Updated final report draft
```
